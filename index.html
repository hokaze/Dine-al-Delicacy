Recent Changes

v0.03 - Bugfixes for combining ingredients
- Fixed inability to select multiple of the same ingredient
- Fixed 3 ingredient delicacies not producing multiple effects
- Fixed bug where it was possible to randomly roll an effect already assigned

v0.02 - Some demoable functionality
- Combining ingredients will add new delicacies, it tracks your ingredient counts, how many of a delicacy you can make, lets you cook a delicacy to remove 1 of each ingredient automatically. Still little visual feedback (everything that happens is reported in console instead), no option for user to choose sub-effect, no renaming, no save/load.

v0.01 - Initial version, early PoC
- very rough mockup, nonfunctional, more of a preview of what some things would look like (and setting up an FF-style menu aesthetic)
